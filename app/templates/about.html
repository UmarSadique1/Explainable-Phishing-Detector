About PhishGuard AI

An intelligent phishing detection system that combines machine learning with explainable AI.

What is PhishGuard AI?

PhishGuard AI is an explainable phishing detection platform designed to help users identify whether an email is likely to be malicious or legitimate. Unlike conventional black-box classifiers, the system not only produces a prediction but also reveals the signals that influenced the result.

The platform uses a machine learning pipeline trained on text data and integrates LIME to highlight the words and phrases that most strongly push a classification towards phishing or legitimacy. This supports both decision-making and user awareness by making model behaviour more transparent.

The wider purpose of the project is to address an important cybersecurity problem while also improving user trust, interpretability, and educational value within phishing detection systems.

How It Works

1

2

3

4

Core Capabilities

AI-Powered Analysis

Uses a trained machine learning model to classify email text based on linguistic patterns associated with phishing behaviour.

LIME Explanations

Highlights influential words and phrases so that users can see which parts of an email pushed the prediction towards phishing or legitimacy.

Human-Readable Output

Presents results in a format that is understandable to non-technical users, supporting both usability and cybersecurity awareness.

Role-Based Access

Supports authenticated access with user and admin roles, enabling the project to evolve into a fuller multi-user security platform.

Explainable Security Design

Focuses not only on prediction accuracy but also on transparency, trust, and interpretability as essential qualities of practical security tools.

Extensible Architecture

The application structure is designed to support future additions such as history, dashboard analytics, threat categories, and richer explanation views.

Why Explainability Matters

Traditional phishing classifiers often behave like a black box: they output a label without showing the reasoning behind it. In a real-world cybersecurity context, this can reduce trust and limit the educational value of the system.

By using explainable AI techniques, PhishGuard AI makes the prediction process more transparent. Users can inspect highlighted words and model signals rather than relying on a label alone. This helps them understand suspicious patterns such as urgency, requests for sensitive information, or deceptive wording that commonly appear in phishing attempts.
